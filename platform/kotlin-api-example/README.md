# Kotlin API Example

This folder contains a standalone Kotlin example project for API calling.

## Purpose

This example demonstrates how to call a Qwen-compatible API in Kotlin.

## Notes

- This example is uploaded as a standalone reference project.
- It is not required to be integrated into the current MobileAIDomestic build.
- Do not commit real API keys into source code.

## Suggested API key setup

Use environment variables such as:

- DASHSCOPE_API_KEY

## Entry

Main source file:

- `src/main/kotlin/Main.kt`
